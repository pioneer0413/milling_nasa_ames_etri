## Reproducibility Impact

- 초기 버전이므로 기존 결과 재현성 영향은 없다.
- 이후 결과 재현을 위해 resolved config, config hash, seed, git state, environment를 저장한다.
- 자동 생성 `experiment_id`는 같은 초 내 병렬 실행 충돌을 막기 위해 microsecond를 포함한다.

## Migration Guide

- 기존 산출물이 없다면 migration은 필요 없다.
- 새 dataset은 loader를 `DATASET_REGISTRY`에 등록하고 필수 metadata schema를 맞춘다.
