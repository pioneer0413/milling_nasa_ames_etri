## Reproducibility Impact

- 초기 버전이므로 기존 결과 재현성 영향은 없다.
- 이후 결과 재현을 위해 resolved config, config hash, seed, git state, environment를 저장한다.
- 자동 생성 `experiment_id`는 같은 초 내 병렬 실행 충돌을 막기 위해 microsecond를 포함한다.
- Domain-shift execution stores per-shift metrics and the final metric as the mean over all six fixed source-target scenarios.
- S1 execution stores per-shift metrics, seed-level six-shift averages, and segment-level mean/std over seeds.
- H1.S1 EDA stores Pearson, Spearman, and mutual information association tables without model training.
- S2 execution stores train-only scaled feature-based prediction results for sensor group combinations and aggregates per-shift metrics into seed-level and 20-seed mean/std summaries.
- H2.S3 execution stores train-only scaled all-sensors feature-based prediction results for feature group combinations and aggregates per-shift metrics into seed-level and 20-seed mean/std summaries.
- H2.S4 execution stores train-only imputed/scaled/encoded process information results and process information addition analyses. Time and material factors are recorded with leakage-review notes because they can act as wear-progression or domain shortcut features depending on deployment timing.

## Migration Guide

- 기존 산출물이 없다면 migration은 필요 없다.
- 새 dataset은 loader를 `DATASET_REGISTRY`에 등록하고 필수 metadata schema를 맞춘다.
